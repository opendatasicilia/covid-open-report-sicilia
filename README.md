# COVID open report Sicilia
Dal 6 ottobre 2021 il Dipartimento AttivitÃ  Sanitarie e Osservatorio Epidemiologico (DASOE) della Regione Siciliana rilascia bollettini settimanali (in formato PDF) ai quali sono allegate due tabelle contenenti dati epidemiologici e vaccinali con dettaglio comunale. Le tabelle allegate vengono convertite in CSV e caricate nella cartella `dati` di questo repo.
